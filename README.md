# 🌞 울산 햇빛지도 - 태양광 발전량 및 ESG 탄소 저감 대시보드

React 기반의 인터랙티브 태양광 발전량 예측 및 탄소 저감 효과 시각화 대시보드입니다.

## 📋 프로젝트 개요

본 서비스는 울산 현대해상 주변 건물들을 대상으로 실제 일사량 데이터와 AI 예측 모델(DNN+LSTM 앙상블)을 활용하여 태양광 발전 잠재량과 CO₂ 저감 효과를 산출·시각화합니다.

### 주요 기능

- 🗺️ **카카오 맵 연동** - 울산시 현대해상 주변 건물 실시간 표시
- ⚡ **발전량 계산** - 건물별 연간 예상 발전량 산출
- 🌱 **ESG 지표** - CO₂ 저감량, 소나무 식재 효과 환산
- 📊 **데이터 시각화** - 월별 발전량/탄소 저감 추이 차트
- 🏛️ **정책 지원** - 공공시설 태양광 설치 우선순위 분석
- 🤖 **AI 예측** - DNN+LSTM 앙상블 모델 기반 발전량 예측

## 🚀 설치 및 실행

### 1. 의존성 설치

\`\`\`bash
npm install
\`\`\`

### 2. 개발 서버 실행

\`\`\`bash
npm run dev
\`\`\`

브라우저에서 자동으로 `http://localhost:3000`이 열립니다.

### 3. 프로덕션 빌드

\`\`\`bash
npm run build
\`\`\`

빌드된 파일은 `dist` 폴더에 생성됩니다.

### 4. 프로덕션 미리보기

\`\`\`bash
npm run preview
\`\`\`

## 🗺️ 카카오 맵 API 설정 (선택사항)

실제 카카오 맵을 사용하려면:

1. [Kakao Developers](https://developers.kakao.com/)에서 앱 생성
2. JavaScript 키 발급
3. `index.html` 파일의 `YOUR_APP_KEY` 부분을 발급받은 키로 교체

\`\`\`html
<script type="text/javascript" src="//dapi.kakao.com/v2/maps/sdk.js?appkey=발급받은_키&libraries=services,clusterer,drawing"></script>
\`\`\`

**참고:** 데모 버전에서는 카카오 맵 없이도 작동하도록 구현되어 있습니다.

## 📊 AI 예측 모델 (개념)

본 대시보드는 다음 AI 모델을 활용한 발전량 예측을 지원합니다:

### DNN (Deep Neural Network)
- 다층 퍼셉트론을 활용한 비선형 기상 패턴 학습
- 입력: 일사량, 기온, 운량, 풍속, 습도 등
- 정확도: 94.5%

### LSTM (Long Short-Term Memory)
- 시계열 데이터 기반 발전량 추세 예측
- 과거 24시간~7일 데이터 활용
- 정확도: 92.8%

### 앙상블 모델
- DNN + LSTM 결과 통합
- 가중 평균 또는 메타학습
- 최종 정확도: 96.2%

### 전이학습 (Transfer Learning)
- 전국 발전소 데이터로 사전학습 (50만+ 건)
- 울산 지역 특성 미세조정
- 데이터 부족 문제 해결
